FAQ


How to add or remove Fonts ?

Add Fonts


To add fonts, Step 1: Copy your .otf font file to randaptcha/assets/fonts directory.

Step 2: Edit settings.php, add a new element to $fonts (array) without extension info

Remove Fonts


Step 1: Go to randaptcha/assets/fonts directory and delete the font you want,
Step 2: Edit settings.php, delete the font name in $fonts (array)

Activate or Deactivate Question Types

Open settings.php, edit $RANDOM_TEXT_QUESTION or $MATH_EQUATION_QUESTION true or false, (true means active, false means deactivate)

Make Math Questions Harder or Easier

Open settings.php, and change the value of $MATH_EQUATION_DIFFICULTY to 1-3 (1 -> Easy, 2 -> Medium, 3-> Hard)
